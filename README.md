# egadd
A program that tracks devices plugged into a computers USB port


| IMPORTANT:    | This is a work in progress |
| ------------- |:-------------:|

### Prerequisites

You will need to have pyudev and pyusb installed

```
$ pip3 install pyudev
$ python3 -m pip install pyusb
```

#### Usage 

```
python3 egadd.py [dev]
```
